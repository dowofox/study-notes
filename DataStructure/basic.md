# 자료구조 (Data Structure)

## 개념
- 컴퓨터에서 처리할 데이터를 효율적으로 관리하고 구조화시키기 위한 방법

---

## 분류

### 단순구조
- 정수 (예 : -1, 0, 1)
- 실수 (예 : 1.2414)
- 문자 (예 : a, b, c)
- 문자열 (예 : abc)

### 선형구조
- 리스트
- 스택
- 큐
- 데큐

### 비선형구조
- 트리
- 그래프

### 파일구조
- 순차 파일
- 색인 파일
- 직접 파일

---

## 자료구조의 이용
- 정렬 (기억장치에 저장된 데이터를 일정한 순서로 나열하는 것)
- 탐색 (기억장치에서 데이터를 찾는 것)
- 파일편성 (데이터를 기억 장치에 저장할 때의 파일 구조)
- 인덱스 (특정 데이터를 빠르게 찾기 위한 것)
